Buttons
Buttons are also called as call to action. We have range of buttons and their states. You may use a tag or button element, you need to add respective classes, and you are good to go.
Example of Primary Button Styles
Whenever you want your user to click on a link or button, use primary style buttons.
hii there power_settings_new power_settings_new
<!-- normal button --> <button class="btn">click</button> <!--primary outlined button --> <button class="btn-outline">click</button> <!-- primary link --> <a href="" class="link-underlined pad-1"> hii there</a> <!-- primary icon button --> <span class="material-icons btn-icn"> power_settings_new </span> <!-- primary outlined icon button --> <span class="material-icons btn-outline-icn"> power_settings_new </span> <!-- button with icon --> <button class="btn"><span class="material-icons "> power_settings_new </span> click </button>
Example of Secondary Button Styles
If you want user not to pay attention to buttons then use Secondary buttons.You just need to put btn-sec after giving the type of button class name in your tag.You can directly copy below html code and use it in your app.
hii there power_settings_new power_settings_new
<!-- secondary btn --> <button class="btn-sec btn">click</button> <!-- secondary outlined button --> <button class="btn-outline-sec btn-outline">click</button> <!-- secondary-link --> <a href="" class="link-underlined link-sec"> hii there</a> <!-- secondary icon button --> <span class="material-icons btn-icn-sec"> power_settings_new </span> <!-- secondary outlined icon button --> <span class="material-icons btn-outline-icn-sec"> power_settings_new </span> <!-- secondary button with icon --> <button class="btn-sec btn"><span class="material-icons "> power_settings_new </span> click</button>
Button with Animations
You can add some small animation on button. you just need to add btn-fancy for fancy primary button, btn-outline-fancy for primary outlined button, link-fancy for fancy link.
hii there
<!-- animated primary button --> <button class="btn-fancy">click</button> <!-- animated primary outlined button --> <button class="btn-outline-fancy">click</button> <!-- animated primary link --> <div><a href="" class=" link-fancy"> hii there</a></div>
